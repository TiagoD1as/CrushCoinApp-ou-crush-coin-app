# Service
Camada de Services com regras de negócio do CrushCoin App

## Estrutura
- **UsuarioService.java** - Regras de negócio para usuários
- **DespesaService.java** - Regras de negócio para despesas
- **InvestimentoService.java** - Regras de negócio para investimentos

## Responsabilidades
- Validação de dados
- Regras de negócio
- Integração com Repository
- Tratamento de erros
