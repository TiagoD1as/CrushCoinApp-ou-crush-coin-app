# Controller
Camada de Controllers REST do CrushCoin App

## Estrutura
- **UsuarioController.java** - Endpoints para gerenciamento de usuários
- **DespesaController.java** - Endpoints para gerenciamento de despesas
- **InvestimentoController.java** - Endpoints para gerenciamento de investimentos

## Métodos REST
- GET - Consultar
- POST - Criar
- PUT - Atualizar
- DELETE - Deletar
