# Entity
Camada de Entidades JPA do CrushCoin App

## Estrutura
- **UsuarioEntity.java** - Entidade JPA de usuários
- **DespesaEntity.java** - Entidade JPA de despesas
- **InvestimentoEntity.java** - Entidade JPA de investimentos

## Tecnologias
- JPA/Hibernate
- Anotações @Entity, @Table, @Id
- Relacionamentos (@OneToMany, @ManyToOne)
