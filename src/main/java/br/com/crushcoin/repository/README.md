# Repository
Camada de Repositories JPA do CrushCoin App

## Estrutura
- **UsuarioRepository.java** - Acesso a dados de usuários
- **DespesaRepository.java** - Acesso a dados de despesas
- **InvestimentoRepository.java** - Acesso a dados de investimentos

## Tecnologia
- Spring Data JPA
- Extends JpaRepository
- Queries automáticas e customizadas
